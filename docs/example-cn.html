2. 行内标记

  • 粗体：**粗体** 或 __粗体__
  • 斜体：*斜体* 或 _斜体_
  • 删除线：~~删除线~~
  • 上标：X2：X<sup>2</sup>
  • 下标：H2O：H<sub>2</sub>O
  • 下划线：
    • u 标签下划线： <u>u 标签下划线</u>
    • ins 标签下划线： <ins>ins 标签下划线</ins>
